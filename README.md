# 📊 Global Steel Market Intelligence Dashboard

## 📌 Project Overview

The **Global Steel Market Intelligence Dashboard** is an interactive Power BI dashboard designed to analyze global steel market trends using commodity prices, import-export data, and macroeconomic indicators. The dashboard helps visualize demand–supply dynamics, price movements, and economic factors that influence the steel industry.

This project simulates a real-world **Market Analyst** workflow by transforming raw market data into meaningful business insights for decision-making.

---

## 🎯 Objectives

- Analyze steel price trends across multiple countries.
- Monitor import and export activities.
- Track commodity prices such as Iron Ore and Coal.
- Study the impact of macroeconomic indicators on steel prices.
- Build an interactive dashboard for market intelligence and reporting.
- Generate business insights using data visualization.
